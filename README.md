# 电路实验互动模拟器

Interactive circuit simulator for physics education

🌐 **在线演示**: https://uongxzh.github.io/circuit-simulator

## 📋 项目简介

这是一个面向中学物理教学的网页版电路模拟器，学生可以通过拖拽方式连接电路元件，实时观察电流、电压变化，直观理解欧姆定律。

## ✨ 功能特色

- 🎨 **拖拽式操作**: 电池、电阻、开关、电表等元件可拖拽连接
- ⚡ **实时计算**: 根据欧姆定律自动计算电流、电压
- 📊 **数据可视化**: 实时显示电流表、电压表读数
- 🧪 **教学友好**: 专为课堂演示和学生实验设计
- 📱 **响应式设计**: 支持平板和电脑使用

## 🚀 使用方法

1. 从元件库拖拽电池、电阻、开关到工作区
2. 点击元件端点进行连线
3. 闭合开关，观察电表读数变化
4. 修改电阻值，观察对电流的影响

## 🎯 教学目标

- 理解串联和并联电路
- 掌握欧姆定律 I = U/R
- 学会使用电流表、电压表
- 探究电阻对电路的影响

## 👨‍🏫 教师使用建议

- 课堂演示：投影展示，实时操作
- 学生实验：每人一台平板，自主探究
- 作业布置：设计特定电路，计算参数
- 在线测试：远程访问，检验理解

## 🛠️ 技术栈

- HTML5 Canvas：绘制电路图
- Vanilla JavaScript：无依赖，易于部署
- CSS3：响应式布局和动画
